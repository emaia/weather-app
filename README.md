Algolia API: https://community.algolia.com/places/  

## Build Setup

```bash
# install dependencies
$ yarn install

# create .env file and edit with api keys
$ cp .env.example .env

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start

```
